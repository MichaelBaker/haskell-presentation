Programming languages are made of primitives, means of combination, and means of abstraction.
In Haskell, functions play all three roles. 
not                          -- Primitive
notAThing = not . isAThing   -- Composition
notAThing "Is this a thing?" -- Abstraction
Here's how we write functions 
  name  args  body
   |     |    _|_
   |     |   |   |
   v     V   V   V
1 square x = x * x
2
3 main = print (square 4)
Associativity 
print square 4 ≡ ((print square) 4)
Conditionals 
1 absoluteValue x | x < 0     = -x
2                 | otherwise = x

Safety of pattern matching 
data ThreeKinds = KindOne   Int
                | KindTwo   [Char]
                | KindThree ThreeKinds

-- Compiler warning (ಥ﹏ಥ)
-- Warning: Pattern match(es) are non-exhaustive
--          In an equation for `show': Patterns not matched: KindThree _
show :: ThreeKinds -> [Char]
show (KindOne a)   = "KindOne " ++ show a
show (KindTwo a)   = "KindTwo " ++ a

-- Noice (⌐■_■)
show :: ThreeKinds -> [Char]
show (KindOne a)   = "KindOne " ++ show a
show (KindTwo a)   = "KindTwo " ++ a
show (KindThree a) = "KindThree (" ++ show a ++ ")"
